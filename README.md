# Krankenkassen CX Plattform

## Installation und Deployment

1. Repository auf GitHub erstellen: `elenawitte85-png/krankenkassen-cx-plattform`
2. Dateien hochladen: `app.py`, `data.csv`, `requirements.txt`, `README.md`
3. Auf https://streamlit.io/cloud anmelden und „New App“ auswählen
4. Repository auswählen: `elenawitte85-png/krankenkassen-cx-plattform`
5. Branch: main, Main file path: app.py
6. Klick auf Deploy → Plattform läuft im Browser
