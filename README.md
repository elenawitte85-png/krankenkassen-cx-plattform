# Krankenkassen CX Plattform – Praxisnahe KPIs

## Deployment

1. GitHub Repository erstellen, z. B.: `elenawitte85-png/krankenkassen-cx-plattform`
2. Dateien hochladen:
   - `app.py`
   - `data.csv`
   - `requirements.txt`
   - `README.md`
3. Auf https://streamlit.io/cloud anmelden
4. "New App" → Repository auswählen
5. Branch: main, Main file path: app.py
6. Klick auf **Deploy**
7. Plattform ist live im Browser
